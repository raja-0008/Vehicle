# Vehicle Tracking & Counting 
A Computer Vision project that detects, tracks, and counts vehicles from traffic videos using OpenCV.

# Features
1. Real-time vehicle detection
2. Vehicle tracking
3. Vehicle counting
4. Traffic monitoring

# Technologies Used
1. Python
2. OpenCV
3. NumPy
4. Computer Vision
